# 🖥️ Computer Architecture & Organization Labs

This repository contains a collection of hardware-level designs and low-level programming projects developed during my Computer Engineering studies. It focuses on CPU design, logic circuits, and MIPS Assembly.

---

## 🏗️ Featured Projects

### 1. 8-Bit Arithmetic Logic Unit (ALU)
A fully functional 8-bit ALU designed using **Logisim**.
* **Operations:** Addition, Subtraction, AND, OR, XOR.
* **Status Flags:** * **Z** (Zero), **N** (Negative), **C** (Carry), **V** (Overflow).
* **Key Concept:** Implementation of 2's complement arithmetic and hardware-level decision making.

### 2. Sequential Logic & Registers
Design and simulation of fundamental memory and timing components:
* **4-Bit Counters:** Up/Down counters using D Flip-Flops.
* **Shift Registers:** Serial-to-Parallel transmission and Binary Multiplication by 2 (Left Shift).

### 3. MIPS Assembly Programming (MARS Simulator)
Low-level software development to understand the hardware-software interface:
* **Algorithms:** Fibonacci sequence, Array processing, and Mathematical Summations.
* **Skills:** Memory allocation (`.data`), Instruction execution (`.text`), and System Calls.

---

## 🛠️ Tools Used
* **Logisim:** Digital logic circuit simulation.
* **MARS (MIPS Assembler and Runtime Simulator):** For Assembly programming and debugging.

## 📚 Learning Outcomes
Through these labs, I gained a deep understanding of:
1. How CPU components (ALU, Registers, Control Unit) interact.
2. The logic behind binary arithmetic and overflow detection.
3. How high-level code is translated into machine instructions.
